Exercise 1: Variables

Create two variables, first_name and last_name, and concatenate them to create a full name.

first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name

Exercise 2: Data Types

Create variables of different data types (string, integer, float, boolean) and print their types and values.

text = "Hello"
number = 42
decimal = 3.14
is_valid = True

print(f"text is '{text}' and has type: {type(text)}")
print(f"number is {number} and has type: {type(number)}")
print(f"decimal is {decimal} and has type: {type(decimal)}")
print(f"is_valid is {is_valid} and has type: {type(is_valid)}")
